# Skeleton

Skeleton adalah komponen untuk menampilkan loading state dengan animasi pulse.

## Contoh

```vue
<Skeleton class="h-40" />
```

::: raw
<Skeleton class="h-40" />
:::

## Height

Untuk mengatur tinggi skeleton gunakan class utility `h-[value]`. Contoh:

```vue
<Skeleton class="h-50" />
```

::: raw
<Skeleton class="h-50" />
:::
